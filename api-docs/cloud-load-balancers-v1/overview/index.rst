About the API
--------------

The Rackspace Cloud Load Balancers service enables customers to quickly
load-balance multiple Cloud Servers or external servers for optimal resource
utilization through a simple Representational State Transfer (REST) web service
interface.

The :how-to:`Cloud Load Balancers FAQ <cloud-load-balancers>` provides more
information about the service.
