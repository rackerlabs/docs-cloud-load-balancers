Resolved issues
~~~~~~~~~~~~~~~

- Fixed an issue that caused a load balancer to go into an ERROR state if error pages 
  were deleted and then SSL termination was subsequently enabled.
- Fixed a concurrency issue that did not allow the deletion of nodes on a load balancer.
- Fixed an issue that caused load balancers created
  before the 1.19.32 release to go into an ERROR state when the connection throttle was
  updated.
